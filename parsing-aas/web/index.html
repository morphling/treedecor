SGLR Parsing as a Service

This service is based on a Java library for layout-sensitive scannerless generalized parsing which is part of SugarJ.
The support for layout-sensitive parsing is soon to be merged with the original JSGLR project.
You might want to consider reading the SDF manual and SDF tutorial provided by the Spoofax project.
This web interface and HTTP API are available as part of the treedecorations project.
Feel free to report bugs and request features on its github page.
